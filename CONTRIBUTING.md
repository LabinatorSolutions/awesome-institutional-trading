# Contributing to Institutional Trading Resources

First off, thank you for considering contributing to this repository! This curated list relies on the community to identify the highest-quality, noise-free resources for institutional order flow and systematic trading.

## 🛑 Core Philosophy & Rules

Before submitting a Pull Request, please ensure your resource adheres strictly to the following rules:

1. **Zero Affiliate Links:** This is a non-negotiable rule. Any PR containing an affiliate link, referral code, or tracking parameter will be immediately rejected and closed.
2. **Institutional Merit:** The resource must teach, support, or apply order flow, volume analysis, market structure, or quantitative/algorithmic methodology.
3. **No Retail Noise:** If a resource relies primarily on lagging indicators (e.g., RSI, MACD), subjective chart patterns, or "get rich quick" signal-based thinking, it does not belong here.
4. **Value Over Sales:** Free resources are preferred. Paid resources (platforms, tools, books) are only included when they represent the absolute best available in their specific category.

## 📝 How to Contribute

1. **Search First:** Check the current `README.md` to ensure the resource isn't already included.
2. **Fork & Branch:** Fork the repository and create a new branch for your addition.
3. **Format:** Follow the existing Markdown table format. Keep descriptions concise, objective, and focused on *why* it matters for an institutional trader.
4. **Submit PR:** Open a Pull Request. In your PR description, briefly explain why this resource meets the criteria above.

## ✅ Format Example

When adding to a table, use the following format:

```markdown
| [Name of Resource](https://link.com) | Author/Creator | Brief, objective description of why it matters. |
```

Thank you for helping keep this repository high-signal and low-noise!
